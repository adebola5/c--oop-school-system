# C# OOP School Management System

## 📌 Description
This is a console-based application built with C# that demonstrates core Object-Oriented Programming (OOP) concepts.

The system allows users to manage different types of people in a school environment, including students and teachers.

---

## 🚀 Features
- Add Students
- Add Teachers
- View all people
- Search for a person by name
- Menu-driven user interaction
- Input validation for numeric values

---

## 🧠 Concepts Used
- **Classes & Objects**
- **Encapsulation**
- **Inheritance**
- **Polymorphism (Method Overriding)**
- Lists and loops
- Basic input validation (`TryParse`)

---

## 🏗️ Project Structure
- `Person.cs` → Base class
- `Student.cs` → Inherits from Person
- `Teacher.cs` → Inherits from Person
- `Program.cs` → Main program logic and menu system

---

## ▶️ How to Run
1. Open the project in Visual Studio
2. Build the solution
3. Run the application
4. Follow the on-screen menu instructions
